Challenge Call

Challenge and Scenario Description

A cocktail party is an interesting scene that manifests itself as the voices of multiple speakers mixed, as if at a raucous social gathering. The challenge in this scene is to find a way to identify the sound source of interest to the listener when multiple sound sources are present at the same time. In recent years, the study of auditory attention decoding (AAD) based on EEG signals has become a very promising solution, which is devoted to decoding the speech/spatial orientation of the listener from the EEG signal in the multi-speaker scene. One of the tasks of auditory attention decoding is to identify and locate the spatial orientation of the speaker in the multi-person speaking environment by analyzing the EEG signal of the listener. Therefore, when an efficient and robust auditory attention decoding algorithm is integrated into the hearing aid device, it can help the hearing impaired to capture the target sound more accurately, ignore the interfering sound, and improve the hearing performance in the multi-speaker scene. Since the existing public datasets are all EEG data of subjects collected in scenes with only auditory stimuli, auditory attention in real scenes is usually accompanied by visual and auditory input at the same time.

We provide an AHU dataset consisting of EEG data collected from 10 subjects each in the audio-only scene, the audio-video scene, who listened to the two-speaker stimulus for an average of 40 minutes, totaling about 13.3 hours of data. In the chinese auditory attention decoding challenge, the team will compete to build the best deep learning model to analyze the spatial orientation (left/right, 0/1) of the subject's attention to the speaker from the EEG signal.

Tracks Overview

Details about the Tracks to be completed

Track 1. Single-subject (SS): The audio-only task and audio-video task are included. The data of each subject is divided into training, validation, and testing according to 8:1:1. The goal of participants is to build a single-subject decoding model, calculate the accuracy for each subject on the test set, and finally get an average decoding accuracy (%).

Track 1.1. audio-only task (SS-AO): Average decoding accuracy (%) for 8 subjects in the audio-only scene.

Track 1.2. audio-video task (SS-AV): Average decoding accuracy (%) for 8 subjects in the audio-video scene.

Track 2. Cross-subject (CS): The audio-only task and audio-video task are included. Each scene gives data on 8 subjects for training and validation, and 2 for testing. The goal of participants is to build a cross-subject decoding model, calculate the decoding accuracy of each subject in the test set, and finally obtain the average decoding accuracy (%).

Track 2.1. audio-only task (CS-AO): Average decoding accuracy (%) for 2 subjects in the audio-only scene.

Track 2.2. audio-video task (CS-AV): Average decoding accuracy (%) for 2 subjects in the audio-video scene.

Data Description

data_for_SS

The zipfile is only suitable for single-subject (SS) tasks and contains data for each subject in each of two scenes (audio-only and audio-video), which have been preprocessed and divided into training and validation sets. The pdf in the zipfile contains a detailed description of this task.

data_for_CS

The zipfile is only suitable for cross-subject (CS) tasks and contains complete data for 8 subjects in each of two scenes (audio-only and audio-video), without dividing the training set and the validation set, and the participants are left with one cross-validation according to their needs. The pdf in the zipfile contains a detailed description of this task.

Evaluation Metric

CorrectTestSamples： The number of test samples in a subject that correctly identify the direction of auditory attention.

TotalTestSamples： The total number of test samples given by a subject.

Subs： The total number of test subjects given by each track.
